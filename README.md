📘 Day 1 – Python Basics
🖨️ Print Output

Use print() to display text or instructions in the console.

print("Hello, world!")
🔤 Strings
Strings are enclosed in " " or ' '
You can combine them using +
Use \n to create a new line
print("Hello" + " World")
print("Line 1\nLine 2")
💬 Comments
Comments start with #
Used to explain code
# This is a comment
print("Hello")
📦 Variables
Used to store data
name = "Hello"
Get the length of a string using len()
print(len(name))
⌨️ User Input
input() gets data from the user (as a string)
user_name = input("Enter your name: ")
print("Hello " + user_name)
⚠️ Common Errors
IndentationError → Incorrect spacing or indentation
SyntaxError → Invalid Python code
